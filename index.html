직장도장

개인정보처리방침 (Privacy Policy)

1. 개요

직장도장(이하 "앱")은 사용자의 개인정보 보호를 중요하게 생각합니다. 본 방침은 앱이 수집하는 정보와 그 사용 방법을 설명합니다.

✅ 본 앱은 모든 데이터를 사용자의 기기 내부에만 저장하며, 외부 서버로 전송하지 않습니다.

2. 수집하는 정보

직장도장은 다음 정보를 기기 내부에 저장합니다:

  • 출퇴근 기록 (날짜, 출근/퇴근 시각, 근무 유형, 메모)
  • 휴가 기록 (연차, 반차, 시간차, 공가, 병가, 육아휴직, 무급휴가, 대체휴가)
  • 연차 설정 (총 연차 일수, 근무시간 설정 등)
  • 사용자 프로필 (닉네임, 직업, 직책, 근무 시작일, 퇴직 목표일)
  • 프로필 사진 (사용자가 직접 선택한 이미지)
  • 커스텀 공휴일 (사용자가 직접 등록한 공휴일)
  • 앱 설정 (테마, 스탬프 설정, 알림 설정, 달력 색상, 통계 항목 순서 등)
  • 앱 최초 설치일

위 정보는 사용자의 기기에만 보관되며, 개발자 또는 제3자에게 자동으로 전송되지 않습니다.

3. 데이터 백업 및 복원

앱은 JSON 파일을 통한 데이터 내보내기/가져오기 기능을 제공합니다:

  • 내보내기 시 출퇴근 기록, 휴가 기록, 연차 설정, 커스텀 공휴일, 앱 설정, 프로필 정보가 JSON 파일로 생성됩니다.
  • 백업 파일에는 무결성 검증을 위한 디지털 서명이 포함되며, 변조된 파일은 가져오기가 거부됩니다.
  • 프로필 사진 파일은 백업에 포함되지 않습니다.
  • 생성된 파일은 사용자가 직접 메일, 클라우드, 기기 저장소 등으로 보관합니다.
  • 백업 파일은 외부 서버에 자동으로 전송되지 않습니다.

데이터 백업은 선택 사항이며, 사용하지 않아도 앱의 모든 기능을 정상적으로 이용할 수 있습니다.

4. 리포트 내보내기

앱은 근무 통계를 이미지(PNG) 또는 PDF 파일로 내보내는 기능을 제공합니다:

  • 내보내기 시 임시 파일이 생성되며, 공유 완료 후 즉시 삭제됩니다.
  • 갤러리에 저장하는 경우 사용자의 사진 라이브러리에 저장됩니다.
  • 리포트에는 닉네임, 근속 기간, 출근 통계가 포함됩니다.

5. 광고

본 앱은 Google AdMob을 통해 배너 광고 및 보상형 광고를 제공합니다. AdMob은 광고 제공 및 최적화를 위해 다음 정보를 수집할 수 있습니다:

  • 광고 식별자 (Android: GAID / iOS: IDFA)
  • 기기 정보 (모델, OS 버전 등)
  • 광고 상호작용 데이터 (노출, 클릭)

iOS 14.5 이상에서는 앱 추적 투명성(ATT) 동의 팝업을 통해 광고 추적 여부를 선택할 수 있습니다. 추적을 허용하지 않아도 앱 사용에 제한이 없습니다.

광고 관련 자세한 내용은 Google 개인정보처리방침을 참고하세요.

6. 권한 사용

앱은 다음 권한을 요청할 수 있습니다:

  • 카메라: 프로필 사진 촬영 시에만 사용
  • 사진/갤러리 접근: 프로필 사진 선택 시에만 사용
  • 갤러리 저장: 근무 리포트 이미지 저장 시에만 사용
  • 알림: 출퇴근 리마인더 알림 사용 시 (선택 사항)
  • 정확한 알람: 예약된 알림 시간에 정확히 전달하기 위해 사용 (Android)
  • 광고 추적: 맞춤 광고 제공을 위해 사용 (iOS, 선택 사항)

모든 권한은 해당 기능 사용 시에만 요청되며, 거부해도 핵심 기능은 정상 작동합니다.

7. 데이터 보관 및 삭제

기기 내 데이터는 앱을 삭제하면 함께 삭제됩니다. 사용자가 내보낸 백업 파일은 사용자가 직접 관리하며, 개발자는 사용자 데이터에 접근하거나 열람할 수 없습니다.

앱 내 설정에서 데이터 초기화 기능을 통해 개별적으로 기록을 삭제할 수도 있습니다.

8. 제3자 제공

앱은 광고 서비스(Google AdMob)를 제외하고 사용자 데이터를 제3자에게 판매하거나 제공하지 않습니다.

9. 아동 개인정보 보호

본 앱은 만 14세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.
